SOS Game

The SOS game is a Java application developed using Object-Oriented Programming. It allows players to play a word-based game where they aim to form "SOS" sequences to earn points. The game features multiple modes, customizable board sizes, and the option to play against a computer. Additionally, it includes a record feature that logs players' moves and turns during the game. Overall, it offers an engaging programming project that demonstrates the use of Java and Object-Oriented principles in game development.

VR Game

In this Dart Shooting VR game, created with Unity and JavaScript, players will experience a simple yet engaging shooting experience in a virtual environment. Using VR controllers, players can grab and shoot objects, while sound effects enhance the overall gaming experience. The game includes a user-friendly UI canvas with instructions and a secondary canvas for scene resets. A score tracker keeps count of target hits, with some targets disappearing upon impact. Overall, this small and immersive VR game promises an enjoyable experience for players to enjoy.

Utility Bill Services Database Management Application

The Energy application is an efficient solution crafted to effectively manage customer information and billing through a connected database server. With its user-friendly interface, the application empowers the company to seamlessly execute essential tasks, such as adding or removing customers and updating their personal details and billing statuses. Furthermore, it provides the flexibility to adjust energy tariffs, accommodating both daily and flat rates, ensuring accurate monthly bill calculations for each customer. The application places a premium on security and accessibility, incorporating a login system that allows authorized company users to interact with the connected database server effortlessly.

Application Submission Portal

TThe GTA Application System modernizes the UMKC Computer Science department's process for collecting, managing, and matching Graduate Teaching Assistant (GTA) applications. It serves two user types: students and administrators.
Users can easily view and apply for jobs, checking application status when logged in. Administrators can create, edit, and manage job postings, including changing application statuses. All data is stored securely in a database.
This full-stack development project employs Node.js, Express.js, MongoDB, and utilizes the EJS extension with layouts for an efficient and user-friendly experience.
